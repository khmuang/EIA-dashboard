 วิธีอัปเดตแบบรวดเร็ว (One-Click Update):
   1. นำไฟล์ Excel ทั้ง 7 ไฟล์ชุดใหม่ มาวางทับไฟล์เดิมในโฟลเดอร์นี้ (ตั้งชื่อไฟล์ให้เหมือนเดิม)
   2. เปิดโปรแกรม Command Prompt (Terminal) ในโฟลเดอร์นี้ ~\.gemini\tmp\project\EIA file (main*)  
   3. พิมพ์คำสั่งด้านล่างนี้แล้วกด Enter:

   1 python update_dashboard.py


  เมื่อสคริปต์ทำงานเสร็จ (ใช้เวลาไม่กี่วินาที) ไฟล์ index.html จะถูกอัปเดตข้อมูล, ตัวเลข, % ความสำเร็จ และกราฟทุกอย่างโดยอัตโนมัติทันทีครับ!
